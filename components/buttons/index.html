Button components

Use BacklotCars custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more. This styleguide includes several predefined button styles, each serving its own semantic purpose, with a few extras thrown in for more control.

Sizes

Use the size prop for additional sizes choosing one of the following values: largest, larger, smaller or smallest.

Interaction states

Buttons will appear pressed or targeted on mouse over (with a darker background and darker border). Theres no need to add a class to the components as they use pseudo-classes.

Inverted state

Invert buttons appearance by adding the inverted boolean attribute to any button component.

Disabled state

Make buttons look inactive by adding the disabled boolean attribute to any button component.

Margins

Button's margin works as any other margin, the prop will expect a string value. You can control individual margins by passing the following string: topMargin(px) rightMargin(px) bottomMargin(px) leftMargin(px).
