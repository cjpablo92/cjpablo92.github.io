Grid

Components and options for laying out, including wrapping containers and grid system.

How it works

BacklotCars’ grid system uses a series of containers, rows, and columns to layout and align content. It’s built with flexbox and is fully responsive. Below is an example and an in-depth look at how the grid comes together.

<BlcContainer margin={size.margin}>
  <BlcRow>
    <BlcColumn col={4}>
      <BlcBox borderWidth={1} />
    </BlcColumn>
    <BlcColumn col={4}>
      <BlcBox borderWidth={1} />
    </BlcColumn>
    <BlcColumn col={4}>
      <BlcBox borderWidth={1} />
    </BlcColumn>
  </BlcRow>
</BlcContainer>

The above example creates three equal-width columns using our predefined grid classes. Those columns are centered in the page with the parent container.

Breaking it down, here’s how it works:

  • Containers provide a means to center and horizontally pad your site’s contents. Use the container component for a responsive pixel width.
  • Rows are wrappers for columns. Each column has horizontal padding (called a gutter) for controlling the space between them. You can control this space by modifying the row's gutter prop.
  • In a grid layout, content must be placed within columns and only columns may be immediate children of rows.
  • Column props indicate the number of columns you’d like to use out of the possible per row. So, if you want three equal-width columns across, you can use col={4}.
  • Column widths are set in calculated pixels, they’re fluid and sized relative to their parent element.

Container

Containers are the most basic layout element but not required when using our default grid system.

While containers can be nested, most layouts do not require a nested container.

Sizes

The default container width is 1280px. Use the size prop for additional widths choosing one of the following values: largest, larger, smaller or smallest. This prop affects the max-width style and wraps the container to the following sizes: largest: 2560px, larger: 1920px, smaller: 960px, smallest: 640px,

Margins

Container's margin works as any other margin but only on the vertical axis, the prop will expect a string value. You cannot control individual margins, just top/bottom.

Examples

<BlcContainer margin="0" size="smallest">
  <BlcBox borderWidth={1} />
</BlcContainer>

Alignment

Use flexbox alignment utilities through alignItems (vertically), alignContent and justifyContent (horizontally) align columns.

Note: thee alignContent prop, alignes the content of the entire row vertically. If the row wrapped into two lines, this property will help you distributing the content.

Vertical alignment

By default columns stretch to fill the height.

Using flex-start columns align to the top.

Using center columns align to the middle.

Using flex-end columns align to the bottom.

Horizontal alignment

By default 'flex-start', columns align to the left.

Using 'center' columns align at the center of the row.

Using 'flex-end' columns align to the right.

Using 'space-between' remaining space will be distributed evenly between columns.

Using 'space-around' remaining space will be distributed as margins to columns.

Using 'space-evenly' remaining space will be spread into offsets and gutters evenly.

Margins

Rows's margin works as container margins, you only target the vertical axis, the prop will expect a string value. You cannot control individual margins, just top/bottom.

Column wrapping

If more than 12 columns are placed within a single row, each group of extra columns will, as one unit, wrap onto a new line unless you change the prop other ways.

Box

A Box is a flexible and extensible content container. It includes options for contextual background and border colors, and powerful display options.

Boxes are built with as little markup and styles as possible, but still manage to deliver a ton of control and customization. Built with flexbox, they offer easy alignment and mix well with other components. They have no margin by default, so use spacing utilities as needed.
